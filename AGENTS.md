# Agent Workflow Rules

1. **No Direct Work on `main`**: Never commit to the `main` branch.
2. **Branch from `dev`**: Always create feature branches from `dev`.
3. **Merge to `dev` First**: All Pull Requests must target `dev`.
4. **Stable `main`**: `main` is only updated from `dev`.
5. **No Visual Polish**: Do not add styling unless requested.
6. **No Fake Logic**: Do not add placeholder code for complex behaviors.
